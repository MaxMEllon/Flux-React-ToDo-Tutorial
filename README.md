## Flux-React-ToDo-Tutorial

### Run with electron

```
$ npm i -g electron-prebuilt
$ npm i
$ electron .
```

### Dependencies

  - [React](https://facebook.github.io/react/)
  - [ReactDOM](https://facebook.github.io/react/)
  - [Flux](https://github.com/facebook/flux)
